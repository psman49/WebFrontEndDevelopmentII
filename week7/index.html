One can use the call() method to set the value of ‘this’ inside a function to an object that is provided as the first argument.
function sayHello(){
return `Hello, my name is ${ this.name }`;}
const clark = { name: 'Clark' };
const bruce = { name: 'Bruce' };
sayHello.call(clark);
<< 'Hello, my name is Clarke'
sayHello.call(bruce);
<< 'Hello, my name is Bruce'
